# Smart Khata Flutter App

This is the Phase 1 Android app for the Smart Khata API.

API base URL is already configured:

```text
https://smart-khata-production.up.railway.app
```

## How to run

This package contains Flutter source files. Because platform folders are generated by Flutter, run:

```bash
cd smart-khata-flutter-app
flutter create .
flutter pub get
flutter run
```

For Android release APK:

```bash
flutter build apk --release
```

## Included screens

- Splash/session check
- Login
- Signup with business creation
- Dashboard summary
- Customers list
- Add customer
- Products list
- Add product/service
- Sales invoice list
- Create invoice
- Logout

## Important

All business APIs use these headers automatically after login:

```text
Authorization: Bearer <token>
x-business-id: <businessPublicId>
```

## Build APK online without installing Flutter

This package includes GitHub Actions workflow:

```text
.github/workflows/build-apk.yml
```

Upload this project to GitHub, open Actions, run **Build Smart Khata Android APK**, then download the APK artifact.

Detailed guide: `GITHUB_BUILD_GUIDE.md`
